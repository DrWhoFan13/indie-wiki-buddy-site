Indie Wiki Buddy is compatible with Firefox for Android! Unfortunately, Firefox for Android currently only supports installing a select few extensions recommended by Firefox staff.

Until Firefox for Android supports more extensions, there is a workaround: Firefox Beta has extended add-on support through its debug menu.

Note that Firefox Beta is a testing platform where new features are added and tested. It's generally reliable, but you may run into bugs and issues as new features are tested. If you are willing to use Firefox Beta (install here), follow these instructions to install Indie Wiki Buddy:

  • Open Firefox Beta, tap on the three dot menu, and select Settings.
  • Tap on "About Firefox Beta".
  • Tap the Firefox Beta logo five times until the "Debug menu enabled" notification appears.
  • Navigate back to Settings. Under "Advanced", you will now see a new entry for "Custom Add-on collection". Click this.
  • Set the collection owner to 17671762, and the collection name to Indie-Wiki-Buddy. Press "ok" to save; Firefox will close.
    • You may also create and maintain your own collection if you wish.
  • Re-open Firefox (if it does re-open on its own) and tap on the three dots menu.
  • Click on "Add-ons". You should now see Indie Wiki Buddy listed. Click the "+" next to the listing to add and activate the extension.
  • When browsing the web, Indie Wiki Buddy's settings menu can be accessed by tapping the three dots, selecing "Add-ons", and clicking "Indie Wiki Buddy".

Hopefully this works for you! If you run into any issues or have any questions, feel free to get in touch via the contact info listed on the homepage.
